## ğŸ”§ How to Rebrand This Template (for clients)

1. Replace the hero image.
2. Update business name + tagline in `Hero.jsx` and `Navbar.jsx`.
3. Edit services list in `Services.jsx`.
4. Update About copy + About image.
5. Adjust colours (via Tailwind classes).
6. Update contact details in `Contact.jsx` + `Footer.jsx`.

This workflow lets you quickly build niche websites for any small business.
